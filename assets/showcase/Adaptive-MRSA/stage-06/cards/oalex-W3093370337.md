# Arbitrary Style Transfer via Multi-Adaptation Network

## Cite_Key
deng2020arbitrary

## Problem
Arbitrary style transfer must preserve detailed content structure from a content image while transferring color tone and stroke patterns from a reference style image. Existing methods often struggle to disentangle and recombine content and style effectively across diverse inputs.

## Method
The paper proposes a Multi-Adaptation Network with two self-adaptation (SA) modules and one co-adaptation (CA) module. The content SA module uses position-wise self-attention to enhance content representation, while the style SA module uses channel-wise self-attention to enhance style representation. The CA module rearranges the distribution of style representation based on content representation using non-local local-similarity calculations. A disentanglement loss is added to better separate main style patterns from exact content structures.

## Data
The abstract mentions various qualitative and quantitative experiments for arbitrary style transfer, but does not specify datasets.

## Metrics
The abstract reports better results than state-of-the-art style transfer methods in qualitative and quantitative evaluations, but gives no metric names or values.

## Findings
The Multi-Adaptation Network improves arbitrary style transfer by better disentangling content and style and recombining them through adaptation modules and disentanglement loss, outperforming prior style transfer approaches in reported evaluations.

## Limitations
This paper is about arbitrary style transfer rather than personalized diffusion-based multi-concept customization, so relevance is methodological rather than task-direct. The abstract omits dataset and metric details.
