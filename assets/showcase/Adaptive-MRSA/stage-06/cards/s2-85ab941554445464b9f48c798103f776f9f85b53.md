# FreeEdit: Mask-Free Reference-Based Image Editing With Multi-Modal Instruction

## Cite_Key
he2024freeedit

## Problem
Reference-based image editing needs to reproduce user-specified visual concepts from a reference image while following natural-language instructions. Existing methods often require manual masks, struggle to reconstruct fine-grained reference details, and lack suitable datasets for reference-based editing with image triplets.

## Method
FreeEdit uses a multi-modal instruction encoder to guide editing without manual masks by implicitly locating editable regions from language instructions. It introduces Decoupled Residual Refer-Attention (DRRA), which injects fine-grained reference features extracted by a detail extractor into the editing process in a residual manner without disturbing the original self-attention. The authors also build FreeBench, a dataset created with a twice-repainting scheme, containing before/after images, instructions, and reference images for tasks such as addition, replacement, and deletion. Training is conducted in phases on FreeBench followed by quality tuning.

## Data
FreeBench, a curated dataset for reference-based image editing containing image triplets with source image, edited target image, instruction, and reference image; it covers object addition, replacement, and deletion.

## Metrics
The abstract states extensive experiments across multiple task types and superiority over existing methods, but does not list named metrics.

## Findings
FreeEdit achieves high-quality zero-shot editing from language instructions while accurately reproducing visual concepts from reference images. Across multiple editing task types, it outperforms existing methods. The new FreeBench dataset addresses a key data bottleneck for reference-based editing evaluation and training.

## Limitations
Unlike FreeCustom, FreeEdit is not tuning-free overall; it relies on phased training on the newly built dataset and quality tuning. The abstract does not provide metric values or cross-dataset generalization details.
