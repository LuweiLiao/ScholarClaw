# FreeCustom: Tuning-Free Customized Image Generation for Multi-Concept Composition

## Cite_Key
ding2024freecustom

## Problem
Customized text-to-image generation methods largely focus on single-concept personalization and struggle with composing multiple user-specified concepts. Existing approaches often require retraining or fine-tuning on a few images per concept, which is slow, deployment-unfriendly, and harder to scale; they also commonly need multiple images to represent each concept.

## Method
FreeCustom is a tuning-free framework for customized image generation with multi-concept composition using only one reference image per concept. Its core components are: (1) Multi-Reference Self-Attention (MRSA), which lets the generation process attend to multiple reference concepts; (2) a weighted mask strategy that encourages the model to focus on relevant reference concepts; and (3) the use of reference images with context interactions, based on the authors’ finding that concepts are preserved better when reference images include contextual interactions.

## Data
Evaluation is described as experiments on both multi-concept composition and single-concept customization settings using one image per concept as input. The abstract does not name specific datasets or benchmarks.

## Metrics
The abstract reports comparisons in terms of concept consistency with references and alignment with input text, and states outperforming or matching training-based baselines on multi-concept composition and single-concept customization. Specific metric names are not provided in the abstract.

## Findings
FreeCustom generates images consistent with the given concepts and better aligned with the input text. It outperforms or performs on par with training-based methods for both multi-concept composition and single-concept customization, while being simpler and avoiding tuning. A key empirical finding is that providing reference images with context interactions improves concept preservation.

## Limitations
The abstract does not specify quantitative metric values, datasets, or failure cases. As a tuning-free method based on reference images, performance may depend on the quality and contextual informativeness of the single reference image per concept.
